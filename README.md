# Terre de Bray - Plateforme E-commerce Bio

Une application web moderne et complète pour la vente de légumes locaux, avec gestion des comptes utilisateurs, panier sécurisé et panneau d'administration avancé.

## 🌟 Nouvelles Fonctionnalités

- **Interface Moderne** : Design épuré avec Tailwind CSS (cartes produits, animations, responsive).
- **Système de Comptes** : Inscription et connexion sécurisées (hachage des mots de passe avec `bcryptjs`).
- **Panier Intelligent** : Réservé aux utilisateurs connectés pour une meilleure gestion des commandes.
- **Panneau Admin Avancé** : 
    - Mise à jour de l'arrivage en texte simple.
    - Ajout de produits un par un avec **téléchargement de photos**.
    - Consultation des commandes reçues.
- **Sécurité** : Accès à la page `admin.html` restreint par email.
